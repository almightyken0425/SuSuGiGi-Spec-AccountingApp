- 即時呈現財務總覽與收支紀錄，並提供快速新增交易與轉帳的入口

## 佈局

### Header

- Header 模式: E
- 篩選按鈕
- Logo
- 搜尋按鈕
- 設定按鈕

### 主內容區

- 資料由 buildPeriodReport 產出
- **期間切換器:**
  - 切換到較舊期間的按鈕
  - 日曆圖示與時間區間標題
  - 切換到較新期間的按鈕
- **環形圖區塊:**
  - 環形圖
    - 同時呈現當期支出與當期收入兩段弧，不受焦點篩選列影響
    - 起點位於環形圖正上方
    - 支出弧自起點向左生長
    - 收入弧自起點向右生長
    - 兩弧長度合計為環形圖一圈，支出弧與收入弧的長度比例對應支出合計與收入合計
    - 兩弧在環形圖底部某點交會，交會位置由兩弧長度比例決定
    - 各側分組依金額由小至大排列，金額較小者鄰近起點，金額較大者鄰近交會點
    - 各側分組色票分配各自依 assignChartColors 規則處理
    - **IF** 僅有支出無收入: 整個環形圖由支出弧填滿
    - **IF** 僅有收入無支出: 整個環形圖由收入弧填滿
    - **IF** 兩者皆無紀錄: 環形圖不顯示
  - 餘額數值，顯示於環形圖中心
- **焦點篩選列:**
  - 兩張等寬焦點卡並排
  - 支出焦點卡
    - 支出圖示
    - 支出合計金額
  - 收入焦點卡
    - 收入圖示
    - 收入合計金額
  - 永遠擇一為啟用樣式，另一為停用樣式
  - 啟用樣式以主色強邊框搭配卡片底色呈現
  - 停用樣式以面板浮起底色呈現

### 逐筆紀錄

- List 模式: Custom
- 含多幣別匯率換算、分組可折疊、類別與日期雙佈局模式
- 資料由 buildPeriodReport 產出
- **分組標題列:**
  - 各分組預設為收合狀態
  - 收合/展開圖示
  - **IF** 類別分組:
    - **IF** 一般類別:
      - 類別圖示底色塊
      - 類別圖示
      - 類別名稱
    - **IF** 轉帳分組:
      - 外部帳戶圖示底色塊
      - 外部帳戶圖示
      - 外部帳戶名稱
  - **IF** 日期分組:
    - 日期文字
  - 該分組小計金額
  - 各分組標題色採統一文字色
- **單筆紀錄列:**
  - 左輔助欄位依分組模式呈現該紀錄在分組標題列以外的維度
  - **IF** 一般交易:
    - **IF** 類別分組:
      - 左輔助欄位顯示日期文字
    - **IF** 日期分組:
      - 左輔助欄位顯示類別圖示
    - 主文字顯示備註
    - 副文字顯示帳戶名稱
    - **IF** 無備註:
      - 主文字不顯示內容
    - 金額
  - **IF** 轉帳紀錄:
    - **IF** 類別分組:
      - 左輔助欄位顯示日期文字
      - 外部帳戶已是分組標題
      - 副文字顯示另一方帳戶名稱
      - 帳戶名稱並列方向箭頭
    - **IF** 日期分組:
      - 左輔助欄位顯示方向圖示
      - 副文字顯示帳戶名稱與外部帳戶名稱，依方向呈現由來源至目的
    - 主文字顯示備註
    - **IF** 無備註:
      - 主文字不顯示內容
    - 金額
  - 單筆紀錄列的類別圖示採統一主色
  - **金額顯示:**
    - 主金額以該筆原始幣別呈現
    - **IF** 幣別模式為多幣別 **AND** 該筆原始幣別不等於 baseCurrencyId 對應幣別:
      - 主金額下方以副文字顯示換算為 baseCurrencyId 對應幣別的金額，前綴 `≈`
    - **ELSE:**
      - 不顯示換算副文字，避免與主金額呈現相同數值
  - **IF** 該筆為定期排程實例（scheduleId 不為 null）:
    - 金額左側並列循環圖示
    - 圖示使用次文字色，視覺權重低於金額
    - 圖示本身不獨立觸發互動，點按列的行為與一般紀錄相同
- **IF** 無紀錄:
  - 不顯示空狀態提示

### Footer

- 新增支出按鈕
- 新增轉帳按鈕
- 新增收入按鈕
- **IF** Undo Bar 啟用:
  - Undo Bar 覆蓋於此，暫時取代新增按鈕
  - Undo Bar 為全域元件，結構與互動見 shared_ui_policies 的 undo_bar_policy

### 載入狀態

- **IF** 初始載入中:
  - 顯示載入文字

## 互動

- **點按報表篩選按鈕:**
  - 導航至 HomeFilterScreen

- **點按搜尋按鈕:**
  - 導航至 SearchScreen

- **點按設定入口按鈕:**
  - 導航至 SettingsScreen

- **點按分組標題列:**
  - 展開或收合該分組

- **點按焦點篩選列任一卡:**
  - **IF** 該卡為當前啟用樣式:
    - 不變更
  - **ELSE:**
    - 該卡切換為啟用樣式
    - 另一卡切換為停用樣式
    - 逐筆紀錄即時切換為對應類型

- **點按單筆紀錄列:**
  - **IF** 一般交易:
    - 帶入紀錄資訊
    - 導航至 TransactionEditorScreen
  - **IF** 轉帳紀錄:
    - 帶入紀錄資訊
    - 導航至 TransferEditorScreen

- **點按新增支出按鈕:**
  - 呼叫 canUserPerformAction，動作識別碼 createTransaction
  - **IF** 回傳禁止:
    - 導航至 PaywallScreen
  - **ELSE:**
    - 帶入當前報表日期
    - 導航至 TransactionEditorScreen，支出模式

- **點按新增收入按鈕:**
  - 呼叫 canUserPerformAction，動作識別碼 createTransaction
  - **IF** 回傳禁止:
    - 導航至 PaywallScreen
  - **ELSE:**
    - 帶入當前報表日期
    - 導航至 TransactionEditorScreen，收入模式

- **點按新增轉帳按鈕:**
  - 呼叫 canUserPerformAction，動作識別碼 createTransfer
  - **IF** 回傳禁止:
    - 導航至 PaywallScreen
  - **ELSE:**
    - 帶入當前報表日期
    - 導航至 TransferEditorScreen

- **水平滑動分頁:**
  - 主內容區支援水平分頁滑動
  - 每頁代表一個時間區間
  - **IF** 時間粒度為全部:
    - 僅顯示一頁，不可滑動
  - 禁止向未來時間區間滑動
  - **IF** 無任何交易或轉帳紀錄:
    - 僅顯示當前時間區間，禁止向過去滑動
  - **IF** 有交易或轉帳紀錄:
    - 可向過去滑動至最早紀錄所在的時間區間
    - 向過去方向滑動時自動載入更多時間區間
  - 時間區間標題跟隨水平滑動

- **向上捲動主內容區:**
  - 環形圖區塊隨主內容區捲動消失

- **捲動回主內容區起點:**
  - 環形圖區塊顯示
