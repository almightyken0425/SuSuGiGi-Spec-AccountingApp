# 登入畫面: LoginScreen

## 畫面目標

- 提供以 Google 帳號登入應用程式的入口

---

## 線框圖

```text
+--------------------------------+
|                                |
|           [品牌標誌]            |
|                                |
|             (G)                |
|                                |
|         條款引導文字            |
|  使用條款連結 · 隱私政策連結     |
|          版權文字               |
+--------------------------------+
```

---

## 佈局

### 品牌區

- 品牌標誌圖形
  - 圖形定案由對側 Design git 的品牌標誌工件承載

### 登入區

- Google 登入按鈕
  - 僅含 Google 圖示，無文字標籤
  - **IF** 載入中:
    - 按鈕顯示載入狀態，停用點擊

### 頁尾

- 條款引導文字
  - 與其下兩個連結相連成完整語意，表達登入即同意該些條款
  - 不重複條款名稱
- 使用條款連結
- 隱私政策連結
- 版權文字

---

## 互動

- **點按 Google 登入按鈕:**
  - 呼叫 handleLogin
  - **IF** 操作中:
    - 顯示載入狀態
  - **IF** 操作成功:
    - 呼叫 resolveLaunchDestination，取得初始落點與付費牆攔截結果
    - 導航至初始落點
    - **IF** 攔截付費牆:
      - 導航至 PaywallScreen
  - **IF** 操作失敗:
    - 顯示登入失敗提示

- **點按使用條款連結:**
  - 開啟外部使用條款頁面

- **點按隱私政策連結:**
  - 開啟外部隱私政策頁面
