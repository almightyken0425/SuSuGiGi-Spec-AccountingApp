# 匯率編輯器: CurrencyRateEditorScreen

## 畫面目標

- 提供新增與更新幣別對匯率的介面

---

## 觸發情境

- 未指定幣別對開啟時為新增模式
- 由匯率列表點選幣別對開啟時為更新模式

---

## 線框圖

```text
+--------------------------------+
| [X]        Set Rate       [✓]  |
+--------------------------------+
|                                |
| [ From Cur ]      [ To Cur ]   |
|                                |
| [ Amount A ]  =  [ Amount B ]  |
|                                |
+--------------------------------+
```

---

## 佈局

### 導覽列

- Header 模式: C
- 關閉按鈕
- 設定匯率 標題
- 完成按鈕
  - **IF** 輸入欄位未填妥:
    - 不可點按

### 幣別對顯示區

- 來源幣別代碼
- 目標幣別代碼
- 目標幣別鎖定為基礎幣別，不可修改
- **IF** 新增模式:
  - 點按來源幣別開啟幣別選擇 modal
- **IF** 更新模式:
  - 來源幣別不可修改

### 幣別選擇 modal

- List 模式: B-1
- modal 內可搜尋的幣別清單
- **IF** 無搜尋結果:
  - 依 list_policy 空狀態顯示

### 匯率輸入區

- 來源金額輸入框
  - 預設值 1
- 等號標籤
- 目標金額輸入框
  - **IF** 更新模式:
    - 預填該幣別對目前匯率

---

## 互動

- **點按關閉按鈕:**
  - 返回上一頁

- **點按 modal 內幣別選項:**
  - 套用為來源幣別
  - 關閉 modal

- **點按完成按鈕:**
  - 呼叫 createCurrencyRate
  - **IF** 操作成功:
    - 返回上一頁
  - **IF** 操作失敗:
    - 顯示錯誤提示
