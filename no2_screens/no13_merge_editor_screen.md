# 合併編輯器: MergeEditorScreen

## 畫面目標

- 提供合併兩個帳戶或兩個類別的介面

---

## 進入參數

- 來源模式：帳戶或類別，由呼叫端於導航時指定

---

## 線框圖

```text
+----------------------------------+
| [X]          合併          [✓]   |
+----------------------------------+
|                                  |
|   [來源圖示名]  ->  [目標圖示名]  |   ← 來源與目標視覺化
|                                  |
|  警告：合併後來源的交易將轉移至    |
|  目標，來源將被刪除。              |
|                                  |
|  +----------------------------+  |
|  | [來源選擇器] -> [目標選擇器] |  |   ← 雙選擇器同框並排（常駐顯示）
|  +----------------------------+  |
|                                  |
+----------------------------------+
```

---

## 佈局

### 導覽列

- Header 模式: C
- 關閉按鈕
- 合併 標題
- 完成按鈕
  - **IF** 來源或目標未選擇:
    - 不可點按
  - **IF** 來源與目標相同:
    - 不可點按

### 來源與目標視覺化

- 來源項目圖示與名稱
- 箭頭
- 目標項目圖示與名稱
  - 以主色強調，表示保留端

### 警告訊息區

- 合併後，所有屬於來源的交易紀錄將轉移至目標，且來源將被刪除

### 雙選擇器區

- 來源選擇器與目標選擇器並排於同一外框內，中間以箭頭表示方向
- 兩選擇器皆常駐顯示，不另彈出選擇清單
- 選擇器視覺對齊 `STATIC_WHEEL_PICKER_TOKENS`
- **IF** 來源與目標相同:
  - 外框以錯誤色標示衝突

---

## 互動

- **點按關閉按鈕:**
  - 返回上一頁

- **來源選擇器:**
  - 列出全部未停用項目，可任選
  - 初始為清單第一項

- **目標選擇器:**
  - 僅列出與來源相容的未停用項目
    - **IF** 類別模式:
      - 與來源同型別
    - **IF** 帳戶模式:
      - 與來源同幣別
  - 初始為相容清單中第一個不等於來源的項目，使預設不與來源相同
  - **IF** 變更來源致目前目標不相容:
    - 目標重置為相容清單中第一個不等於來源的項目
  - **IF** 相容清單僅來源一項:
    - 目標維持為來源，由衝突狀態呈現

- **來源與目標相同時:**
  - 完成按鈕不可點按
  - 雙選擇器外框以錯誤色標示衝突

- **點按完成按鈕:**
  - **IF** 帳戶模式:
    - **IF** 來源與目標幣別不同:
      - 顯示錯誤提示，中止流程
    - 呼叫 mergeAccounts
  - **IF** 類別模式:
    - 呼叫 mergeCategories
  - **IF** 操作成功:
    - 呼叫 UndoLogic.showUndo
    - 返回上一頁
  - **IF** 操作失敗:
    - 顯示錯誤提示
