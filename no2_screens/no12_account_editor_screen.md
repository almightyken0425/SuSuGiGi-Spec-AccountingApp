# 帳戶編輯器: AccountEditorScreen

## 畫面目標

- 提供新增或編輯帳戶的統一介面

---

## 線框圖

```text
+--------------------------------+
| [X]         Title         [✓]  |
+--------------------------------+
|       [ Name Input ]           |
| Currency: [ TWD - New Tw.. ] v |
| Type: [現金][銀行][信用卡]..   |
| Icon: [ ▣ ][ ◊ ][ ◇ ][ ○ ]    |
|       [ ◯ ][ □ ][ △ ][ ▽ ]    |
|       [ ... ]                  |
|                                |
| Disable: [ Switch ]            |
|                                |
|        [Delete Account]        |
+--------------------------------+
```

---

## 佈局

### 導覽列

- Header 模式: C
- 關閉按鈕
- 標題
  - **IF** 新增模式:
    - 新增帳戶
  - **IF** 編輯模式:
    - 編輯帳戶
- 完成按鈕
  - **IF** 必填欄位未填妥:
    - 不可點按

### 欄位區

- 帳戶名稱輸入框
- 幣別選擇器
  - List 模式: B-1
  - 模式適用展開後的清單
  - **IF** 收合:
    - 顯示當前所選幣別與展開指示器
    - 顯示格式為「代碼 - 名稱」對偶，與 BaseCurrencySettingScreen 對齊
  - **IF** 展開:
    - 搜尋輸入框
    - 幣別列表（每項顯示「代碼 - 名稱」對偶格式）
  - **IF** 編輯模式:
    - 顯示為停用樣式
    - 不可修改
- 標準帳戶類型選擇器
  - 五個選項常駐顯示，單選
  - 選項：現金、銀行帳戶、信用卡、投資、其他
  - **IF** 新增模式:
    - 預設選中清單第一項
    - 可點按切換
  - **IF** 編輯模式:
    - 可點按切換
- 圖示選擇器
  - List 模式: 不適用
  - 純 icon grid，未套用 list 政策
  - 圖示網格常駐顯示，無折疊狀態
  - 點按圖示即時更新預覽

### 操作區

- **IF** 編輯模式:
  - 停用此帳戶 開關
  - 刪除此帳戶 按鈕

---

## 互動

- **點按關閉按鈕:**
  - 返回上一頁

- **點按幣別選擇器:**
  - 展開或收合幣別列表
  - **IF** 展開:
    - 搜尋輸入框自動聚焦

- **輸入幣別搜尋文字:**
  - 即時篩選幣別列表

- **選擇幣別列表項目:**
  - 收合選擇器並更新顯示

- **點按標準帳戶類型選項:**
  - **IF** 按下選項為當前所選類型:
    - 忽略，不執行任何動作
  - **ELSE:**
    - 切換為新類型

- **選擇圖示:**
  - 即時更新預覽

- **點按完成按鈕:**
  - **IF** 新增模式:
    - 呼叫 canUserPerformAction，動作識別碼 createAccount
    - **IF** 回傳禁止:
      - 導航至 PaywallScreen
    - **IF** 選擇外幣:
      - 呼叫 canUserPerformAction，動作識別碼 useForeignCurrency
      - **IF** 回傳禁止:
        - 導航至 PaywallScreen
    - 呼叫 createAccount
    - **IF** 操作成功:
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示
  - **IF** 編輯模式:
    - 呼叫 updateAccount
    - **IF** 操作成功:
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示

- **點按刪除按鈕:**
  - 顯示刪除確認對話框
  - **IF** 確認刪除:
    - 呼叫 deleteAccount
    - **IF** 操作成功:
      - 呼叫 UndoLogic.showUndo
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示
