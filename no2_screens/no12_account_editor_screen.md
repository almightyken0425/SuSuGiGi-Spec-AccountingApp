# 帳戶編輯器: AccountEditorScreen

## 畫面目標

- 提供新增或編輯帳戶的統一介面

---

## 線框圖

```text
+--------------------------------+
| [X]         Title         [✓]  |
+--------------------------------+
|       [ Name Input ]           |
| Currency: [ TWD - New Tw.. ] v |
| Icon: [ ▣ ][ ◊ ][ ◇ ][ ○ ]    |
|       [ ◯ ][ □ ][ △ ][ ▽ ]    |
|       [ ... ]                  |
|                                |
| Disable: [ Switch ]            |
|                                |
|           [Delete]             |
+--------------------------------+
```

---

## 佈局

### 導覽列

- Header 模式: C
- 關閉按鈕
- 標題
  - **IF** 新增模式:
    - 新增帳戶
  - **IF** 編輯模式:
    - 編輯帳戶
- 完成按鈕
  - **IF** 必填欄位未填妥:
    - 不可點按

### 欄位區

- 帳戶名稱輸入框
- 幣別選擇器
  - List 模式: B-1
  - 模式適用展開後的清單
  - **IF** 收合:
    - 顯示當前所選幣別與展開指示器
    - 幣別顯示 `alphabeticCode` 與 `name`，與 BaseCurrencySettingScreen 對齊
  - **IF** 展開:
    - 搜尋輸入框
    - 幣別列表，每項顯示 `alphabeticCode` 與 `name`
  - **IF** 編輯模式:
    - 顯示為停用樣式
    - 不可修改
- 圖示選擇器
  - List 模式: 不適用
  - 純 icon grid，未套用 list 政策
  - 圖示網格常駐顯示，無折疊狀態
  - 點按圖示即時更新預覽

### 操作區

- **IF** 編輯模式:
  - 停用此帳戶 開關
  - 刪除 按鈕
    - 標籤依 delete_button_policy 規範

---

## 互動

- **點按關閉按鈕:**
  - 返回上一頁

- **點按幣別選擇器:**
  - 展開或收合幣別列表
  - **IF** 展開:
    - 搜尋輸入框自動聚焦

- **輸入幣別搜尋文字:**
  - 即時篩選幣別列表

- **選擇幣別列表項目:**
  - 收合選擇器並更新顯示

- **選擇圖示:**
  - 即時更新預覽

- **點按完成按鈕:**
  - **IF** 新增模式:
    - 呼叫 createAccount
    - **IF** 操作成功:
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示
  - **IF** 編輯模式:
    - 呼叫 updateAccount
    - **IF** 操作成功:
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示

- **點按刪除按鈕:**
  - 顯示刪除確認對話框
  - **IF** 確認刪除:
    - 呼叫 deleteAccount
    - **IF** 操作成功:
      - 呼叫 showUndo
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示
