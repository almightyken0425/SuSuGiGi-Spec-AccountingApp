# 搜尋: SearchScreen

## 畫面目標

- 提供搜尋所有交易與轉帳備註的介面，並作為導航至紀錄編輯畫面的入口

---

## 觸發情境

- 以 Modal 呈現
- 由點擊首頁搜尋入口觸發

---

## 線框圖

```text
+----------------------------------+
| [X]         搜尋                 |
+----------------------------------+
| .------------------------------. |
| | [Icon] Category     Amount   | |
| |                    ≈ Conv    | |
| |        Note(hl)         Date | |
| |------------------------------| |
| | [Icon] Category  [⟳] Amount  | |
| |        Note(hl)         Date | |
| '------------------------------' |
+----------------------------------+
| [ Search...                    ] |
+----------------------------------+
```

---

## 佈局

### 導覽列

- Header 模式: D
- 關閉按鈕
- 搜尋 標題

### 搜尋輸入區

- Search 模式: Bottom Pill

### 搜尋結果區

- List 模式: Custom
- 含關鍵字 highlight、多幣別匯率換算、雙列佈局
- 視覺對齊對側 Design git 的 TX_LIST_TOKENS 群組
- **IF** 搜尋框為空:
  - 顯示 "輸入關鍵字以搜尋交易" 提示
- **IF** 無符合結果:
  - 顯示 "找不到結果" 提示
  - 副標回顯當前搜尋關鍵字
- 搜尋結果列表
  - 外層由群組卡片包覆
  - row 間以 hairline 分隔線分隔
  - 交易與轉帳各自最多顯示 50 筆
  - 依日期由新至舊排序
- **單筆紀錄列:**
  - 雙列佈局
  - **上列:**
    - 左：類別圖示置於 outline 框內
    - 中：類別名稱或轉帳標籤
    - 右：金額
  - **下列:**
    - 左：備註，關鍵字以主色高亮
    - 右：日期
  - 金額顯色：支出採錯誤色、收入採成功色、零值採主文字色
  - **IF** 該筆為定期排程實例，即 scheduleId 不為 null:
    - 金額左側並列循環圖示
    - 圖示使用次文字色，視覺權重低於金額
    - 圖示本身不獨立觸發互動，點按列的行為與一般紀錄相同
  - **IF** 幣別模式為多幣別 **AND** 該筆原始幣別不等於 baseCurrencyId 對應幣別:
    - 主金額下方以副文字顯示換算為 baseCurrencyId 對應幣別的金額，前綴 `≈`
  - **ELSE:**
    - 不顯示換算副文字，避免與主金額呈現相同數值

---

## 互動

- **點按關閉按鈕:**
  - 關閉 Modal

- **輸入搜尋文字:**
  - 延遲 300 毫秒後自動觸發搜尋，更新搜尋結果列表

- **點按搜尋結果列表項目:**
  - **IF** 交易紀錄:
    - 導航至 TransactionEditorScreen
  - **IF** 轉帳紀錄:
    - 導航至 TransferEditorScreen
