# 轉帳編輯器: TransferEditorScreen

## 畫面目標

- 提供新增與編輯轉帳記錄的介面，並作為建立定期轉帳排程的入口

---

## 線框圖

```text
+--------------------------------+
| [X]       Transfer        [✓]  |
+--------------------------------+
|  Date: Oct 25, 2024     [Recur]|
+--------------------------------+
| +----------------------------+ |
| |  From $  →  To $           | |  ← 金額 group box（同框包 from / to）
| +----------------------------+ |
| +----------------------------+ |
| |  From Acct  →  To Acct     | |  ← 帳戶 group box（同框包 from / to）
| +----------------------------+ |
| [Note Input]                   |
|                                |
|           [Delete]             |
+--------------------------------+
| [ Calculator Keypad ]          |  ← absolute dock，含 ⌫ 鍵
+--------------------------------+
```

---

## 佈局

### Header

- Header 模式: C
- 關閉按鈕
- 轉帳 標題
- 完成按鈕
  - **IF** 必填欄位未填妥:
    - 不可點按
  - **IF** 轉出與轉入帳戶相同:
    - 不可點按

### 日期選擇區

- 日期選擇器模式: Calendar Dialog · Datetime
- 定期切換按鈕
  - **IF** 此筆轉帳屬於某排程:
    - 啟用視覺狀態
  - **ELSE:**
    - 停用視覺狀態

### 定期設定區

- **IF** 定期設定區為展開:
  - 嵌入 RecurringOptions

### 金額輸入區

- 同一 group box 包住：
  - 轉出金額輸入框
  - 箭頭
  - 轉入金額輸入框
- **IF** 同幣別轉帳:
  - 轉入金額輸入框為停用狀態
  - 轉入金額自動跟隨轉出金額

### 帳戶選擇區

- 同一 group box 包住：
  - 轉出帳戶選擇器
  - 箭頭
  - 轉入帳戶選擇器
- **IF** 轉出與轉入帳戶相同:
  - group box 外框轉錯誤狀態色

### 備註區

- 備註輸入框

### 操作區

- **IF** 編輯模式:
  - 刪除 按鈕
    - 標籤依 delete_button_policy 規範

### 計算機鍵盤區

- 計算機鍵盤
  - 數字鍵與小數點鍵
  - 加減乘除鍵與等號鍵
  - 退格鍵
- **IF** 轉出或轉入金額輸入框聚焦:
  - 顯示計算機鍵盤
- **IF** 備註輸入框聚焦:
  - 收起計算機鍵盤
  - 浮現系統鍵盤

## 互動

- **進入畫面:**
  - **IF** 編輯模式:
    - 依此筆轉帳的當前內容顯示各欄位
    - **IF** 此筆轉帳屬於某排程:
      - 當前定期規則為該排程的頻率、間隔、結束日期
    - **IF** 此筆轉帳不屬於任何排程:
      - 當前定期規則為空
  - **IF** 新增模式:
    - 當前定期規則為空
  - 定期設定區預設為收合
  - 轉出金額輸入框預設聚焦

- **點按關閉按鈕:**
  - 返回上一頁

- **點按定期切換按鈕:**
  - 呼叫 canUserPerformAction，動作識別碼 createRecurringTransaction
  - **IF** 回傳禁止:
    - 導航至 PaywallScreen
  - **ELSE:**
    - 切換定期設定區的展開或收合狀態

- **點按轉出金額輸入框:**
  - 轉出金額輸入框聚焦
  - 顯示計算機鍵盤

- **點按轉入金額輸入框:**
  - **IF** 跨幣別轉帳:
    - 轉入金額輸入框聚焦
    - 顯示計算機鍵盤
  - **IF** 同幣別轉帳:
    - 無反應

- **點按計算機鍵盤:**
  - 按鍵結果即時顯示於聚焦的金額輸入框

- **點按完成按鈕:**
  - **IF** 此筆轉帳屬於某排程:
    - 顯示定期編輯模式對話框
    - **IF** 選擇僅此一筆 或 此筆及未來:
      - 呼叫 updateSchedule
  - **IF** 不屬於任何排程且定期規則已設定:
    - **IF** 新增模式:
      - 呼叫 createSchedule
    - **IF** 編輯模式:
      - 原一般轉帳轉為排程
      - 呼叫 deleteTransfer
      - 呼叫 createSchedule
  - **IF** 不屬於任何排程且定期規則為空:
    - **IF** 新增模式:
      - 呼叫 canUserPerformAction，動作識別碼 createTransfer
      - **IF** 回傳禁止:
        - 導航至 PaywallScreen
      - **ELSE:**
        - 呼叫 createTransfer
    - **IF** 編輯模式:
      - 呼叫 updateTransfer
  - **IF** 操作成功:
    - 返回上一頁
  - **IF** 操作失敗:
    - 顯示錯誤提示

- **點按刪除按鈕:**
  - **IF** 此筆轉帳屬於某排程:
    - 顯示定期刪除模式對話框
    - **IF** 選擇僅此一筆 或 此筆及未來:
      - 呼叫 deleteSchedule
      - **IF** 操作成功:
        - 呼叫 showUndo
  - **ELSE:**
    - 顯示刪除確認對話框
    - **IF** 確認刪除:
      - 呼叫 deleteTransfer
      - **IF** 操作成功:
        - 呼叫 showUndo
  - **IF** 操作成功:
    - 返回上一頁
  - **IF** 操作失敗:
    - 顯示錯誤提示
