# 週起始日設定: WeekStartSettingScreen

## 畫面目標

- 提供選擇一週起始日的介面

---

## 線框圖

```text
+--------------------------------+
| [X]       Week Start       [✓] |
+--------------------------------+
| 跟隨語系                     ✓  |
| 週日                            |
| 週一                            |
+--------------------------------+
```

---

## 佈局

### 導覽列

- Header 模式: C
- 關閉按鈕
- 週起始日 標題
- 完成按鈕

### 週起始日列表

- List 模式: B-1
- 週起始日選項
  - 選項名稱：跟隨語系、週日、週一
  - 跟隨語系 代表依使用者語系慣例決定
  - **IF** 為目前選取選項:
    - 顯示選取標記

---

## 互動

- **點按關閉按鈕:**
  - 返回上一頁

- **點按週起始日選項:**
  - 將此選項標為目前選取選項

- **點按完成按鈕:**
  - 呼叫 setWeekStart
  - 返回上一頁
