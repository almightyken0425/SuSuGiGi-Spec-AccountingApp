# 時區設定: TimeZoneSettingScreen

## 畫面目標

- 提供選擇 App 使用時區的介面

---

## 線框圖

```text
+--------------------------------+
| [X]        Time Zone       [✓] |
+--------------------------------+
| Taipei (UTC+08:00)          ✓  |
| Tokyo (UTC+09:00)              |
| New York (UTC-05:00)           |
| ...                            |
+--------------------------------+
| [ Search...                  ] |
+--------------------------------+
```

---

## 佈局

### 導覽列

- Header 模式: C
- 關閉按鈕
- 時區 標題
- 完成按鈕

### 搜尋列

- Search 模式: Bottom Pill

### 時區列表

- List 模式: B-1
- 進入畫面時目前選取時區置頂；其後選取其他時區不重新排序，順序維持至離開畫面
- 時區項目
  - 城市名稱
  - UTC 偏移
  - **IF** 為目前選取時區:
    - 顯示選取標記

---

## 互動

- **點按關閉按鈕:**
  - 返回上一頁

- **輸入搜尋文字:**
  - 依時區名稱或城市名稱即時篩選列表

- **點按時區項目:**
  - 將此項目標為目前選取時區

- **點按完成按鈕:**
  - 呼叫 setTimeZone
  - 返回上一頁
