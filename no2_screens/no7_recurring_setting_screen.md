# 定期設定區: RecurringOptions

## 畫面目標

- 提供嵌入於交易與轉帳編輯器中的定期規則設定區，控制此筆是否設為定期，以及定期的頻率、間隔與結束條件

---

## 觸發情境

- 嵌入於 TransactionEditorScreen 與 TransferEditorScreen，由所在畫面的定期觸發按鈕展開或收合

---

## 線框圖

```text
+--------------------------------+
| Frequency                      |
| [Daily][Weekly][Monthly][Yearly]
|                                |
| Every                          |
| [  1  ] Month(s)               |
|                                |
| End                            |
| [Never] [On Date] [Date Picker]|
+--------------------------------+
```

---

## 初始狀態

- 由所在畫面傳入當前定期規則, 規則可為空
- **IF** 規則為空:
  - 啟用開關: 關
  - 設定內容區: 不可操作, 顯示元件預設值
- **IF** 規則非空:
  - 啟用開關: 開
  - 設定內容區: 可操作, 依規則內容顯示頻率、間隔、結束條件

---

## 佈局

### 標題列

- 區域標題
- 啟用開關

### 設定內容區

- **IF** 啟用開關為關:
  - 設定內容區不可操作
- **IF** 啟用開關為開:
  - 設定內容區可操作

- 頻率選項列
  - 每日選項
  - 每週選項
  - 每月選項
  - 每年選項

- 間隔列
  - 間隔輸入框
  - 單位文字
    - **IF** 頻率為每日:
      - 顯示 Day
    - **IF** 頻率為每週:
      - 顯示 Week
    - **IF** 頻率為每月:
      - 顯示 Month
    - **IF** 頻率為每年:
      - 顯示 Year

- 結束條件選項列
  - 永不選項
  - 於指定日期選項
  - 結束日期選擇器
    - 結束日期選擇器模式: Date-only
    - 顯示日期初始值: 既有規則的結束日期; 若無結束日期則為今天
    - 可選任意過去或未來日期, 無上下限
    - **IF** 結束條件為於指定日期:
      - 可操作
    - **IF** 結束條件為永不:
      - 不可操作

---

## 互動

- **切換啟用開關為開:**
  - 啟用設定內容區並回傳目前規則至所在畫面

- **切換啟用開關為關:**
  - 停用設定內容區並回傳清空狀態至所在畫面

- **點按頻率選項:**
  - 更新當前頻率為點按之選項

- **修改間隔輸入:**
  - 僅接受數字字元，非數字字元不更新
  - 開頭為 0 時不更新
  - 以輸入的數字更新當前間隔

- **點按永不選項:**
  - 結束條件設為永不, 清除結束日期
  - 結束日期選擇器顯示的日期保留, 不變

- **點按於指定日期選項:**
  - 結束條件設為於指定日期
  - 結束日期設為結束日期選擇器當下顯示的日期

- **變更結束日期選擇器顯示的日期:**
  - 更新結束日期選擇器顯示的日期為新值
  - **IF** 結束條件為於指定日期:
    - 結束日期同步更新為新值
  - **IF** 結束條件為永不:
    - 結束日期維持為空
