# 主要貨幣設定: BaseCurrencySettingScreen

## 畫面目標

- 提供選擇 App 主要貨幣的介面

---

## 線框圖

```text
+--------------------------------+
| [X]      Base Currency     [✓] |
+--------------------------------+
| USD - United States Dollar  ✓  |
| TWD - New Taiwan Dollar        |
| JPY - Japanese Yen             |
| ...                            |
+--------------------------------+
| [ Search...                  ] |
+--------------------------------+
```

---

## 佈局

### 導覽列

- Header 模式: C
- 關閉按鈕
- 主要貨幣 標題
- 完成按鈕

### 搜尋列

- Search 模式: Bottom Pill

### 貨幣列表

- List 模式: B-1
- 進入畫面時目前選取貨幣置頂；其後選取其他貨幣不重新排序，順序維持至離開畫面
- 貨幣項目
  - `alphabeticCode`
  - `name`
  - **IF** 為目前選取貨幣:
    - 顯示選取標記

---

## 互動

- **點按關閉按鈕:**
  - 返回上一頁

- **輸入搜尋文字:**
  - 依貨幣代碼或名稱即時篩選列表

- **點按貨幣項目:**
  - 將此項目標為目前選取貨幣

- **點按完成按鈕:**
  - 呼叫 setBaseCurrency
  - 返回上一頁
