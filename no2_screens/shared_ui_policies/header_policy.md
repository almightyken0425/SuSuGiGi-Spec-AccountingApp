# Header Policy 導航列共用元件政策

## 政策定位

- 本 module 內所有 screen 共用的導航列政策
- 統一所有 screen 的導航列模式與按鈕詞庫
- screen spec 只標模式代號，不重述按鈕細節
- 視覺定案由對側 Design git 的 `10_foundations/` 承載
- 實作元件與平台技術細節由對側 Impl git 的 `CLAUDE.md` UI Coding Guideline 承載

---

## 模式分類

- **模式 A — Stack-List 列表頁:**
    - **適用:**
        - 設定主頁
        - 類別清單
        - 帳戶清單
        - 貨幣清單
        - 匯率清單
        - 偏好設定
        - 資料管理
        - 各類選擇器
    - **元件條目順序:**
        - 原生返回
        - 標題
        - 動作元件
    - **動作元件條件:**
        - 通常為空
        - 列表類可放新增
        - 列表類可放合併

- **模式 B — Stack-Editor 編輯頁:**
    - **適用:**
        - 預留
        - 目前無 screen 採用
    - **元件條目順序:**
        - 原生返回
        - 動態標題
        - 動作元件留空

- **模式 C — Modal-Form 表單型 modal:**
    - **適用:**
        - 交易編輯器
        - 轉帳編輯器
        - 帳戶編輯器
        - 類別編輯器
        - 合併編輯器
        - 匯率編輯器
        - 貨幣明細設定
        - 主題設定
        - 啟動模式設定
        - 主要貨幣設定
        - 語言設定
        - 時區設定
    - **元件條目順序:**
        - 關閉動作
        - 標題
        - 完成動作
    - **完成動作狀態:**
        - 表單必填欄位未滿足時設為 disabled

- **模式 D — Modal-Info 純展示 modal:**
    - **適用:**
        - 首頁篩選
        - 付費牆
        - 搜尋頁
    - **元件條目順序:**
        - 關閉動作
        - 標題
        - 動作元件留空
    - **搜尋頁特例:**
        - 搜尋輸入框依 search_policy 規範採對應模式
    - **付費牆特例:**
        - header bar 僅關閉動作
        - 不設畫面標題

- **模式 E — Special-Dashboard 主畫面儀表板:**
    - **適用:**
        - 首頁
    - **元件條目順序:**
        - 篩選動作
        - 標題或品牌名
        - 搜尋動作
        - 設定動作

- **模式 F — Modal-Wizard 多步驟精靈:**
    - **適用:**
        - 匯入精靈
    - **元件條目順序:**
        - 左動作
        - 動態標題
        - 右動作
    - **左動作條件:**
        - 首步為關閉
        - 其餘步驟為返回
    - **右動作條件:**
        - 末步為送出
        - 其餘步驟為前進
        - 當前步驟驗證未通過時設為 disabled
    - **導航承載:**
        - 步驟前進與返回由 header 左右動作承載
        - 不設置底導航列

---

## 按鈕詞庫

- **絕對禁止:**
    - header 自訂按鈕含任何文字
    - 含中文文字
    - 含英文文字
    - 例外為標題本身
- **強制解法:**
    - 按鈕以 icon 表意、不含文字
    - 動作與 icon 的對應由對側 Design git 的 ACTION_ICON_MAP 承載

---

## screen spec 引用方式

- **要求:**
    - 各 screen spec 在 Header 佈局區域開頭標註 `Header 模式: A` 此類代號
    - 不重述按鈕詞庫
    - 不重述視覺定案

- **合規寫法:**

```
### Header

- Header 模式: C
- 關閉動作
- 標題
  - **IF** 支出:
    - 支出
  - **IF** 收入:
    - 收入
- 完成動作
  - **IF** 必填欄位未填妥:
    - 不可點按
```
