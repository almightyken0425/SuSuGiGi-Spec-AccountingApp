# Date Picker Policy 日期選擇器共用元件政策

## 政策定位

- 本 module 內所有需要選擇日期或日期時間的 screen 共用政策
- 統一觸發器與選擇 dialog 的結構與互動
- screen spec 只標模式代號，不重述 dialog 結構
- 視覺定案由對側 Design git 的 `10_foundations/` 與 `20_components/` 承載
- 實作元件、平台 API、套件選用由對側 Impl git 的 `CLAUDE.md` UI Coding Guideline 承載

---

## 模式分類

- **模式 Datetime — 日期加時間:**
    - **適用:**
        - 交易編輯器的記錄日期
        - 轉帳編輯器的記錄日期
    - 觸發器顯示日期與時間文字
    - dialog 含日期選擇與時間選擇
- **模式 Date-only — 純日期:**
    - **適用:**
        - 定期設定的結束日期
    - 觸發器顯示日期文字
    - dialog 僅含日期選擇，無時間選擇

## 互動規則

- **點按觸發器 pill:**
    - 彈出 dialog
    - 進入日選擇子模式

- **日選擇子模式:**
    - **點按標題列:**
        - 切換至月選擇子模式
    - **日期格上下滑捲動:**
        - 單月一頁，切換顯示的月份，放手吸附到整頁
    - **點按某一日:**
        - 選定該日
    - **轉動時間滾輪，僅模式 Datetime:**
        - 選定時與分

- **月選擇子模式:**
    - **點按標題列:**
        - 切換回日選擇子模式
    - **月份格上下滑捲動:**
        - 單年一頁，切換顯示的年份，放手吸附到整頁
    - **點按某一月:**
        - 選定該月
        - 停留於月選擇子模式

- **點按 dialog 外部:**
    - 關閉 dialog

---

## 行為契約

- **即時生效:**
    - 任一選擇動作立即反映到呼叫畫面的日期時間值
    - 無確認或取消按鈕

- **顯示格式:**
    - 觸發器與標題列文字依使用者語系格式顯示
    - 星期列文字依使用者語系顯示
    - 週起始日依 `Settings` 的 `weekStart` 偏好決定；值為 `auto` 時依使用者語系慣例
    - 時間依使用者偏好以 24 小時制或 12 小時制顯示

- **月曆格範圍:**
    - 月曆格僅顯示當月日期
    - 相鄰月位置留白，無日期、不可點

- **可選範圍:**
    - 由呼叫畫面指定上下限
    - 未指定時無限制
    - 超出範圍的日期不可選

## screen spec 引用方式

- **要求:**
    - 各 screen spec 在日期選擇段落標註模式代號
    - 不重述觸發器與 dialog 結構
    - 不重述視覺定案

- **合規寫法:**

```
### 日期選擇區

- 日期選擇器模式: Datetime
```

```
- 結束日期選擇器模式: Date-only
```
