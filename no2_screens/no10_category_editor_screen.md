# 類別編輯器: CategoryEditorScreen

## 畫面目標

- 提供新增或編輯類別的統一介面

---

## 線框圖

```text
+--------------------------------+
| [X]         Title         [✓]  |
+--------------------------------+
| Type: [ 支出 v ][ 收入 v ]     |
| Name: [ Input Text ]           |
| Map To: [ Standard Cat ] v     |
| Icon: [ Icon Preview ] v       |
|                                |
| Disable: [ Switch ]            |
|                                |
|           [Delete]             |
+--------------------------------+
```

---

## 佈局

### 導覽列

- Header 模式: C
- 關閉按鈕
- 標題
  - **IF** 新增模式:
    - 新增類別
  - **IF** 編輯模式:
    - 編輯類別
- 完成按鈕
  - **IF** 必填欄位未填妥:
    - 不可點按

### 欄位區

- 類別類型選擇器
  - 兩按鈕橫排，單選
  - 選項：支出、收入
  - **IF** 新增模式:
    - 預設選中支出
    - 可點按切換
  - **IF** 編輯模式:
    - 顯示為停用樣式
    - 不可點按切換
    - 避免改變已建立類別的類型導致歷史交易紀錄分類不一致
- 類別名稱輸入框
- 標準類別映射選擇器
  - List 模式: B-1
  - 模式適用展開後的標準類別清單
  - **IF** 收合:
    - 顯示當前所選標準類別名稱與展開指示器
  - **IF** 展開:
    - 搜尋輸入框
    - 標準類別列表
- 圖示選擇器
  - List 模式: 不適用
  - 純 icon grid，未套用 list 政策
  - **IF** 收合:
    - 顯示當前所選圖示與展開指示器
  - **IF** 展開:
    - 圖示網格

### 操作區

- **IF** 編輯模式:
  - 停用此類別 開關
  - 刪除此類別 按鈕

---

## 互動

- **點按關閉按鈕:**
  - 返回上一頁

- **點按類別類型按鈕:**
  - **IF** 編輯模式:
    - 忽略，不執行任何動作
  - **IF** 新增模式:
    - **IF** 按下按鈕為當前所選類型:
      - 忽略，不執行任何動作
    - **ELSE:**
      - 切換為新類型
      - 重設標準類別映射為新類型的預設標準類別
      - 重設圖示為新類型的預設圖示
      - 保留類別名稱

- **點按標準類別映射選擇器:**
  - 展開或收合標準類別列表
  - **IF** 展開:
    - 搜尋輸入框自動聚焦

- **輸入標準類別搜尋文字:**
  - 即時篩選標準類別列表

- **選擇標準類別列表項目:**
  - 收合選擇器並更新顯示

- **點按圖示選擇器:**
  - 展開或收合圖示網格

- **選擇圖示:**
  - 收合選擇器並更新預覽

- **點按完成按鈕:**
  - **IF** 新增模式:
    - 呼叫 canUserPerformAction，動作識別碼 createCategory
    - **IF** 回傳禁止:
      - 導航至 PaywallScreen
    - **ELSE:**
      - 呼叫 createCategory
      - **IF** 操作成功:
        - 返回上一頁
      - **IF** 操作失敗:
        - 顯示錯誤提示
  - **IF** 編輯模式:
    - 呼叫 updateCategory
    - **IF** 操作成功:
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示

- **點按刪除按鈕:**
  - 顯示刪除確認對話框
  - **IF** 確認刪除:
    - 呼叫 deleteCategory
    - **IF** 操作成功:
      - 呼叫 UndoLogic.showUndo
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示
