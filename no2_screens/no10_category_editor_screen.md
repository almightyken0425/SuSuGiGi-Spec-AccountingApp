# 類別編輯器: CategoryEditorScreen

## 畫面目標

- 提供新增或編輯類別的統一介面

---

## 線框圖

```text
+--------------------------------+
| [X]         Title         [✓]  |
+--------------------------------+
|       [ Name Input ]           |
| 收支: [ 支出 ] v               |
| Icon: [ ▣ ][ ◊ ][ ◇ ][ ○ ]    |
|       [ ◯ ][ □ ][ △ ][ ▽ ]    |
|       [ ... ]                  |
|                                |
| Disable: [ Switch ]            |
|                                |
|           [Delete]             |
+--------------------------------+
```

---

## 佈局

### 導覽列

- Header 模式: C
- 關閉按鈕
- 標題
  - **IF** 新增模式:
    - 新增類別
  - **IF** 編輯模式:
    - 編輯類別
- 完成按鈕
  - **IF** 必填欄位未填妥:
    - 不可點按

### 欄位區

- 類別名稱輸入框
- 類別類型選擇器
  - List 模式: B-1
  - 模式適用展開後的類型清單
  - 選項：支出、收入
  - **IF** 收合:
    - 顯示當前所選類型名稱與展開指示器
  - **IF** 展開:
    - 類型列表
  - **IF** 新增模式:
    - 預設選中支出
    - 可修改
  - **IF** 編輯模式:
    - 顯示為停用樣式
    - 不可修改
    - 避免改變已建立類別的類型導致歷史交易紀錄分類不一致
- 圖示選擇器
  - List 模式: 不適用
  - 純 icon grid，未套用 list 政策
  - 圖示網格常駐顯示，無折疊狀態
  - 圖示網格為 `tags` 含 `category` 的 `IconDefinitions` 清單
  - 預設選取顯示清單的第一個圖示
    - **IF** 清單為空:
      - 選取 fallback 圖示
  - 點按圖示即時更新預覽

### 操作區

- **IF** 編輯模式:
  - 停用此類別 開關
  - 刪除 按鈕
    - 標籤依 delete_button_policy 規範

---

## 互動

- **點按關閉按鈕:**
  - 返回上一頁

- **點按類別類型選擇器:**
  - **IF** 編輯模式:
    - 忽略，不執行任何動作
  - **IF** 新增模式:
    - 展開或收合類型列表

- **選擇類型列表項目:**
  - **IF** 編輯模式:
    - 忽略，不執行任何動作
  - **IF** 新增模式:
    - **IF** 選擇項目為當前所選類型:
      - 收合選擇器
    - **ELSE:**
      - 切換為新類型
      - 重設選取圖示為圖示網格的第一個
        - **IF** 清單為空:
          - 重設為 fallback 圖示
      - 保留類別名稱
      - 收合選擇器

- **選擇圖示:**
  - 即時更新預覽

- **點按完成按鈕:**
  - **IF** 新增模式:
    - 呼叫 createCategory
    - **IF** 操作成功:
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示
  - **IF** 編輯模式:
    - 呼叫 updateCategory
    - **IF** 操作成功:
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示

- **點按刪除按鈕:**
  - 顯示刪除確認對話框
  - **IF** 確認刪除:
    - 呼叫 deleteCategory
    - **IF** 操作成功:
      - 呼叫 showUndo
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示
