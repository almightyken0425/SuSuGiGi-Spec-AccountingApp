# 啟動畫面: LaunchScreen

## 畫面目標

- 應用程式啟動期間呈現品牌基礎態，待登入狀態確定後轉場至登入或首頁

---

## 線框圖

```text
+--------------------------------+
|                                |
|                                |
|                                |
|             $wish              |
|                                |
|                                |
|                                |
+--------------------------------+
```

---

## 佈局

### 品牌區

- App 名稱文字
  - 位置、字級、字重、顏色對齊 LoginScreen 品牌區的 App 名稱
  - 啟動期間為畫面唯一元素
- 背景
  - 對齊應用程式內頁底色，使轉場至 LoginScreen 或 HomeScreen 時底色不跳變

---

## 互動

- 啟動畫面無使用者互動，轉場由 presentLaunchTransition 驅動
- **IF** 未登入:
  - 導航至 LoginScreen
- **IF** 已登入:
  - 導航至 HomeScreen
